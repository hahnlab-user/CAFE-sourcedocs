§ cafe_cmd

CafeShellCommand cafe_cmd[]

Holds the list of commands that are available in Cafe.

Each element consists of a command and the function that is called to handle that command. Terminated by a NULL, NULL pair Functions include cafe_cmd_lambda, cafe_cmd_family, cafe_cmd_tree, etc.
